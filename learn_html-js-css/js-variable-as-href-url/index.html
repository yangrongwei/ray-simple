The <a> tag defines a hyperlink, which is used to link from one page to another.

a href="URL"
Possible values:

o An absolute URL - points to another web site (like href="http://www.example.com/default.htm")
o A relative URL - points to a file within a web site (like href="default.htm")
o Link to an element with a specified id within the page (like href="#top")
o Other protocols (like https://, ftp://, mailto:, file:, etc..)
o A script (like href="javascript:alert('Hello');")
